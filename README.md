# KuvaNurkka
```
Web application for uploading and browsing images, audio and videos.
```

## Features
```
-Upload images, audio and video
-Browse
-Favorite
-Leave comments
-Login/logout
-Search
-Browse by type
-See own and other users' profiles: all the files they have uploaded 
```
